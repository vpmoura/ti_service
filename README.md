# NAC 1 - Spring MVC FIAP

### Nome do projeto: ti_service

* Primeira avaliação continuada da matéria de microsserviços do 3° ano do curso de Sistemas de Informação da faculdade FIAP.
* O projeto consiste na implementação de conceitos da estrutura mvc utilizando recursos do framework Springboot.
* É composto por um Sistema de gerenciamento de chamados. Apresentando uma tela inicial contendo um botão que redireciona o usuário para a tela de cadastro de solicitação, uma segunda tela de abertura de solicitação e uma terceira tela que mostrará a confirmação da abertura de solicitação, sendo os dados de cadastro persistidos no banco de dados.
